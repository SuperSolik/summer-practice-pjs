Для запуска алгоритма нажмите кнопку "Start Animation" для пошаговой демонстрации работы алгоритма с автоматическим переключением шагов, скорость переключения регулируется слайдером "Скорость анимации".
Для ручного переключения шагов алгоритма нажмите кнопку ">" для перехода на шаг вперед или "<" для перехода на шаг назад.
Для быстрого получения результата работы алгоритма (перехода на последний шаг) нажмите ">>". Для возврата в начало (первый шаг) нажмите "<<".
Граф можно импортировать из файла при помощи меню File->import graph или сочетания клавиш Ctrl+O. Аналогично возможен экспорт файл при помощи File->export graph или сочетания Ctrl+S.
Модификация графа:
Нажатием правой кнопки мыши на панель графа открывается всплывающее меню, в котором возможно создание вершины с заданным именем и полная очистка графа.
Нажатием правой кнопки мыши на вершину графа открывается всплывающее меню, в котором возможно добавление и удаление ребер между вершинами, переименование вершины и её удаление.
